# Как добавить новую запись
<!-- position: 2 -->

Войдите в **Панель управления** и нажмите **Новая запись**.

1. Заполните поле **Название**.
2. Заполните поле **Содержимое**.
3. Нажмите кнопку **Опубликовать**.
4. Готово!
